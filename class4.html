HTML Forms

How to collect user input

A Form Example


    <form action="http://foo.com" method="post">

      <label for="firstname">First name</label>
      <input type="text" id="firstname" name="firstname">

      <button type="submit">Send my greetings</button>

    </form>

The Form Tag

Marks the start and end of an HTML form for collecting user input.


    <form action="http://foo.com" method="post">


    </form>

Sending and retrieving form data

Input

A multi-type element used for capturing user input.

Often called a "form control".


      <input type="text" id="firstname" name="username">

Some types:

  • Text (default)
  • Checkbox
  • Radio button
  • Range
  • Search
  • Password

HTML Input Docs

Labels

Provide a label for an input using a for and id attributes.

Describes an input so a screen reader can identify it.

Provides a larger tap/focus area.


    <label for="firstname">First name</label>
    <input id="firstname" type="text">

More info on form labels

Textarea

A multi-line element for typing paragraphs of text.


      <textarea name="description" rows="10" cols="50">
      </textarea>

Buttons

Used to trigger changes in an interface.

Use the type attribute to specify a button type: submit, reset, or button. Defaults to button.


    <button>Edit Profile</button>

What's Next?

Once you've got your pages, you'll need somewhere to host them.

Option 1: Web Hosting

  1. Register a domain name
  2. Choose a web host
  3. Update name servers for domain name (DNS)
  4. FTP files from your machine to server
  5. Enjoy!

Getting started

For a small personal site...

  • Domain name registration should not cost more than $15/yr
  • Hosting should not cost more than $60/yr
  • Hosting choices may depend on supported server-side tech, security needs

Vendors

Domain registration

  • PowerPipe
  • NameCheap
  • Not GoDaddy

Hosting

  • InMotion
  • DreamHost ($$, an investment)
  • Digital Ocean (DevOps skills/curiosity required)

FTP - File Transfer Protocol

FTP: A process used to copy files from one machine to another.

FTP client: The software you use to do that.

FTP Clients

Mac

  • CyberDuck
  • Transmit

PC

  • FileZilla
  • Windows Explorer
