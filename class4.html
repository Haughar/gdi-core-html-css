Float

  • "Floating" an element takes it in the normal flow, as far to the left or right of it's containing element as possible.
  • Any other elements, such as paragraphs or lists, will wrap around the floated element.
  • Always specify a width when floating an element, otherwise the element is likely to take up the whole page and not appear floated.
  • You can specify a) whether an element is floated or not, and b) which side it ﬂoats on.

Using floats to place elements side by side

If you want two block level elements to be side by side, you need to float both elements. One left, and one right.

Clear

  • Clearing tells the element on which side (right, left, or both) other elements cannot appear.
  • If you had an image floated left, and you did not want the paragraph to appear next to it, you would add clear: left; to the paragraph.
  • Clearing both sides makes sure ﬂoats don’t ﬂow past the clear element.

  clear: both; 

  /* 
  or: 
  clear: left;

  or:
  clear: right;
  */

Labels

Provide a label for a form element using a for and id attributes.

Describes an input so a screen reader can identify it.

Provides a larger tap/focus area.


    <label for="firstname">First name</label>
    <input id="firstname" type="text">

More info on form labels

Input

A multi-type element used for capturing user input.

Often called a "form control".


      <input type="text" id="firstname" name="firstname">

Some types:

  • Text (default)
  • Checkbox
  • Radio button
  • Range (Slider)
  • Search
  • Password

What's Next?

Once you've got your pages, you'll need somewhere to host them.

Option 1: Web Hosting

  1. Register a domain name
  2. Choose a web host
  3. Update name servers for domain name (DNS)
  4. FTP files from your machine to server
  5. Enjoy!

Getting started

For a small personal site...

  • Domain name registration should not cost more than $15/yr
  • Hosting should not cost more than $60/yr
  • Hosting choices may depend on supported server-side tech, security needs
